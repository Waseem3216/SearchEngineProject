# XYMZ Retro Search Engine

Open `index.html` in a browser, or use VS Code Live Server.

## Implemented features

- Centered XYMZ logo above the search bar
- Early-2000s portal styling with bevels, panels, badges, marquee, scanlines, visitor counter, and web ring
- In-page `XYMZ Results` window
- Search modes: Web, Images, News, Videos, Wikipedia, and Maps
- Loading/search progress meter
- Sound effects toggle powered by Web Audio
- Theme switcher
- Search history and saved searches using localStorage
- `Spin Web Wheel` random search
- `I'm Feeling Retro`
- XYMZ AnswerBot
- Add Your Site form
- Guestbook
- Web Buddy popup
- Mouse sparkle effects
- Improved About, Help, and Corporate Services pages

## Notes

Sound effects require clicking the Sound FX button because browsers block audio until user interaction.
Search result cards open live search pages in new tabs when clicked.
